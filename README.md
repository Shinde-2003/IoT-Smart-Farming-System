# IoT-Smart-Farming-System
An IoT-based smart farming solution developed in Cisco Packet Tracer for monitoring environmental conditions and automating irrigation using smart sensors and network-connected devices.

## Overview
The IoT Smart Farming System is a Cisco Packet Tracer project that demonstrates the use of Internet of Things (IoT) technology in modern agriculture. The system uses smart sensors and network-connected devices to monitor environmental conditions and automate farming operations.

## Features
- Soil moisture monitoring
- Temperature monitoring
- Automated irrigation control
- Real-time sensor data collection
- Wireless communication between IoT devices
- Smart agriculture automation

## Technologies Used
- Cisco Packet Tracer
- IoT Sensors
- Wireless Network
- Smart Devices

## Project Objectives
- Improve farming efficiency through automation
- Reduce water wastage
- Monitor environmental conditions in real time
- Demonstrate IoT applications in agriculture
